# Deep_Learning_assignment
A repo for the assignments of course Deep Learning at Uppsala University

1. pre-course assignment
    Dataset: datasets/Auto.csv
    Packages needed: numpy matplotlib pandas
    How to run the code:
        1) cd pre-assignment
        2) python main.py
    Files generated: 
        * cost1.png
        * cost2.png
        * prediction.png
